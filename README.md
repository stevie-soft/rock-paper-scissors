# KPO (Kő Papír Olló) online egyszemélyes játék

## Működés

A jólismert kő-papír-olló játék webes alkalmazásként megvalósítva HTML, CSS, és JavaScript segítsével. Célja a szórakoztatás, a felhasználó feladata legalább három kört nyerni a számítógép ellen.

## Érdekesség

Az visszajelzésekbe ékelt hangulatjelek a nyerések/döntetlenek/vesztések számától függenek és aszerint változnak.

## Háttér

A program személyes célra, projektfeladat megvalósítása céljából készült.
